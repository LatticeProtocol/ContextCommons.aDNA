# Stage 6: Implementation & Reporting

## Purpose

Deliver the program while meeting all reporting obligations. This stage spans the full grant period (typically 12 months). The file remains here until the grant reaches its final reporting period.

## Processing Instructions

1. Track program delivery against grant commitments
2. Collect data continuously: participant demographics, attendance, pre/post assessments, outcomes
3. Submit reports on schedule:
   - **Foundation**: interim and final narrative + financial reports
   - **Federal**: SF-425 financial reports (quarterly/semi-annual), performance progress reports
4. Document participant stories and collect photos (with signed release forms)
5. Track spending against approved budget categories — flag variances early
6. For federal grants: maintain time-and-effort documentation for all grant-funded staff

## Quality Gate (Exit Criteria)

Before advancing to `07_closeout/`:

- [ ] All interim reports filed on time
- [ ] Spending within approved budget categories (or modifications approved)
- [ ] Outcome data collected and compiled
- [ ] No outstanding compliance issues
- [ ] Final report period reached

## Advancement

Move file to `07_closeout/` when entering the final 2 months of the grant period. Update `stage: closeout`.
