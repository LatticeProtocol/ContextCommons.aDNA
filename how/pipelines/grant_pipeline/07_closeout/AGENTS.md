# Stage 7: Closeout

## Purpose

File final reports, reconcile funds, document outcomes, and initiate renewal conversations. This stage also holds opportunities that were declined, withdrawn, or not pursued — providing a complete record of all pipeline activity.

## Processing Instructions

### For Completed Grants

1. Submit final narrative and financial reports per funder requirements
2. Reconcile all expenditures — ensure every dollar is accounted for
3. Return any unexpended funds per grant agreement terms
4. Document outcomes for CC's organizational records (use for future proposals)
5. Begin renewal conversation if funder accepts repeat applications
6. If renewing: create a new file in `01_prospect/` with `source_ref` linking to this grant

### For Declined/Withdrawn Opportunities

Files that exit the pipeline before award land here with one of these stage values:
- `declined_to_apply` — CC chose not to apply after qualifying
- `loi_declined` — funder did not invite full application
- `application_declined` — funder did not award
- `withdrawn` — CC withdrew the application

Document any funder feedback. These records inform future applications.

## Quality Gate (Exit Criteria)

For completed grants:
- [ ] Final reports submitted and accepted by funder
- [ ] All funds reconciled
- [ ] Outcomes documented in CC organizational records
- [ ] Renewal decision made (pursue or close)

## File Retention

All files in this directory are permanent records. Standing Order #6: archive, never delete.
