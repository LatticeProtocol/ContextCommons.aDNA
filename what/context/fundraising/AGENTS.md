# what/context/fundraising/ — Agent Guide

## What's Here

Fundraising and grant strategy context for the Context Commons program — how CC funds itself to serve communities. These files cover the funding landscape, grant writing guidance, curated catalogs of aligned programs, operational procedures, and fiscal sponsorship models. They are loaded for any work involving grant applications, partnership proposals, budget planning, or sustainability strategy.

## Subtopics

| File | Tokens | Purpose |
|------|--------|---------|
| `context_cc_fundraising_strategy.md` | ~2.5K | Diversification model, Year 1-2 targets, portfolio stacking, framing principles |
| `context_cc_fundraising_grant_writing.md` | ~2K | CC-specific framing, funder positioning, LOI/application components |

| `context_cc_fundraising_fiscal_sponsorship.md` | ~2K | Model A vs Model C, relevant sponsors, 501(c)(3) transition pathway |

**Total**: ~14K tokens

## Load/Skip Decision

**Load this topic when**:
- Writing grant applications, LOIs, or funding proposals
- Planning fundraising strategy or evaluating new funding opportunities
- Creating partnership materials that reference funding structure
- Setting up fiscal sponsorship or compliance infrastructure
- Budgeting for pilots or program operations

**Skip when**:
- Working on curriculum content or context ingestion
- Building forge wrappers or website infrastructure
- Detailed pilot logistics that don't involve budget
- Technical aDNA or lattice work

**Typical load**: Strategy file (~2.5K) + one catalog file (~2.5K) + grant writing (~2K) = ~7K for a focused grant session. Full topic load (~14K) only for comprehensive fundraising planning.

**Token cost**: ~14K for full topic, ~2-2.5K per individual file.
