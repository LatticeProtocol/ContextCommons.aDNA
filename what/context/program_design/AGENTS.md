# what/context/program_design/ — Agent Guide

## What's Here

Foundational context for the Context Commons program — the thesis, core principles, program architecture, and governance model. These files establish the "why" and "what" of the program. They are the first context loaded for any work involving program identity, mission alignment, or external communication.

## Load/Skip Decision

**Load this topic when**:
- Creating any external-facing content (website, presentations, partnership proposals)
- Designing curriculum content (the thesis informs pedagogy)
- Onboarding new contributors or community partners
- Making program-level decisions that affect mission or scope

**Skip when**:
- Working on purely technical infrastructure (forge setup, CI/CD)
- Detailed curriculum module authoring (load `curriculum_framework/` instead)
- Operational tasks within an established campaign

**Token cost**: ~8K for full topic, ~2K per individual file.
