# who/communities/ — Agent Guide

## What's Here

Pilot community profiles — one file per community participating in or being evaluated for the Context Commons program. Each profile documents the community's infrastructure, leadership, demographics, needs, and pilot status.

## Working Rules

- **Naming**: `community_{descriptive_name}.md` (underscores, lowercase)
- **Required frontmatter**: `type: community`, `created`, `updated`, `status`, `last_edited_by`, `tags`, `location`, `population`, `infrastructure`, `steward_contact`, `pilot_status`
- **Check before overwriting**: Read `updated` field — if edited today by someone else, confirm with user
- **Set audit trail**: Update `last_edited_by` and `updated` on every edit
- **Community voice gate**: Content about a community must be reviewed by someone from that community before publishing (Standing Order #7)
- **Link to stewards**: Each community should link to its steward profiles via `[[stewards/steward_name]]`

## Load/Skip Decision

**Load this directory when**:
- Planning or evaluating pilot programs
- Creating community-facing materials
- Looking up infrastructure or contact details for a specific community

**Skip when**:
- Working on curriculum content that is not community-specific
- Working on forge infrastructure or technical tooling

**Token cost**: ~300 tokens (this AGENTS.md). Individual records are ~200-400 tokens each.
