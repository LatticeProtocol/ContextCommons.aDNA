# Anthropic

## Overview

Anthropic is a prospective technology partner with uniquely deep alignment to the Context Commons. The program is built on **Claude Code** as the primary agentic development environment, the **aDNA Context Graph Standard** is designed to work with Claude's context window and tool-use capabilities, and Anthropic's stated commitment to AI safety and beneficial AI maps directly to the program's community-governance model. This is not an abstract partnership — the program's core toolchain runs on Anthropic's infrastructure.

## Organization Profile

- **Organization:** Anthropic PBC
- **Type:** Corporate — AI safety and research (Public Benefit Corporation)
- **Mission alignment:** Anthropic's mission ("the responsible development and maintenance of advanced AI for the long-term benefit of humanity") aligns directly with the Context Commons thesis: AI systems should serve communities under community governance. The program demonstrates what "beneficial AI" looks like in practice — not in a research paper, but in a church in Grand Rapids and a school in Los Angeles.
- **Geographic presence:** San Francisco (HQ); distributed engineering

## Partnership Type & Contributions

- **Partnership type:** API access, compute, technical mentorship, curriculum input
- **Contribution type:**
  - **Claude API access** — subsidized or donated API credits for community-operated agent systems and curriculum development
  - **Claude Code licenses** — the program's primary development environment; community/education tier pricing or grants
  - **Technical mentorship** — Anthropic engineers contributing to Lattice mentor network, curriculum review, and safety guardrail design
  - **Curriculum collaboration** — input on responsible AI practices, prompt engineering best practices, agent safety patterns
  - **Compute grants** — cloud compute for development and community systems
- **Scale:** API credits for 2 pilot sites (~20-30 active developers + community systems); Claude Code access for all youth builders and stewards; mentorship hours from engineering team

## Terms & Non-Negotiables

- Community governance is preserved — Anthropic contributes technical resources and safety expertise, not program direction
- No data extraction — community context graphs and conversation data belong to communities; Anthropic does not acquire rights to community-generated context for training
- Open standards — the aDNA ontology and Lattice Protocol remain open; the program teaches agentic system design principles, not a single vendor's toolchain (though Claude is the primary tool)
- Safety alignment — Anthropic's participation reinforces, not replaces, community-defined ethical guardrails; the Curriculum Council establishes what agents should and should not do

## What We Offer Anthropic

- **A concrete model of beneficial AI** — communities directing Claude-powered agent systems under democratic governance, with real outcomes for real people
- **Edge deployment insights** — how Claude performs in resource-constrained, multilingual, intergenerational contexts that don't look like a San Francisco office
- **Responsible AI demonstration** — an auditable, governed, community-owned program Anthropic can reference when asked "what does beneficial AI actually look like?"
- **Youth pipeline** — program graduates who understand agentic systems design, context engineering, and responsible AI practices
- **Curriculum feedback** — what works and what doesn't when teaching prompt engineering, context graphs, and agent design to non-technical communities
